# نظام المحاسبة المتكامل

نظام محاسبي شامل مكتوب بـ HTML، CSS، و JavaScript يتضمن:
- إدارة الفواتير
- إدارة المخزون
- إدارة العملاء والموردين
- التقارير المالية
- شجرة الحسابات

## المميزات
- واجهة مستخدم عربية بالكامل
- تصميم متجاوب
- سهل الاستخدام
- تقارير تفصيلية
- إدارة كاملة للمخزون

## التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Font Awesome
- Chart.js

## كيفية التشغيل
1. قم بتحميل المشروع
2. افتح ملف `index.html` في المتصفح
3. ابدأ باستخدام النظام
